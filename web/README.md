# Web App

## Getting Started

Bootstrap Environment settings:

```
export NODE_ENV=[?]
       STRAVA_CLIENT_SECRET=[?]
       POSTMARK_SECRET=[?]
       SESSION_SECRET=[?]
       MONGODB_URI=[?]
```

Run the development server:

```bash
npm run dev
# or
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser.
